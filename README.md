# Iris Classification Mini Project

This project uses Python and Scikit-learn to classify Iris flowers using different classification algorithms.

## Project Description

The goal of this project is to import the Iris dataset, train different classification models, compare their accuracy, and compare their predicted and actual outputs using confusion matrices.

The classification models used are:

```text
Logistic Regression
KNN Classification
Support Vector Machine
Decision Tree Classification
Random Forest Classification
Gaussian Naive Bayes Classification
